# Amir Horiquini

**`Bioinformática / Tecnologias Ômicas`**

Me chamo Amir Horiquini, Biomédico & Nutricionista, Mestre em Biociências pela USP, Doutorando em Biotecnologia com ênfase em saúde. Sou apaixonado por tecnologia e compartilho aqui um pouco do meu trabalho voltado à análise de dados em Bioinformática, com foco em análise de RNAseq voltada a dados de fungo-hospedeiro.
